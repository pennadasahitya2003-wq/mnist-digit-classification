# MNIST Digit Classification

## Description
This project uses Deep Learning to classify handwritten digits (0-9) from the MNIST dataset.

## Technologies Used
- Python
- TensorFlow/Keras (or PyTorch)
- NumPy

## Dataset
MNIST handwritten digit dataset containing 60,000 training images and 10,000 testing images.

## Features
- Image preprocessing
- Neural Network model training
- Digit prediction
- Model evaluation
